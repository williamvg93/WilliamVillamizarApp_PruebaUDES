Instrucciones:

1) Crear base de datos en Mysql, abrir el archivo UdesDDL.txt, en el se encuentra el DDL de la base de datos.
2) Abrir el proyecto WilliamVillamizarApp con visual studio code.
3) Para ejecutar el programa utilizar el siguiente comando en el terminal: dotnet watch run -p .\ApiUdes\
	Con esto se ejecutará el servidor y nos abrirá en el navegador una interfaz(swagger) en donde se podrán probar
	los endpoint que se crearon.
